Overlay app

App can show some other window or webview over most fullscreen games.

Usefull for single monitor users. Can be used to show chats, maps, game helps over the game.

How to use app

  1. Start app
  2. Go to url you need. position webview window
  3. Press ALT+W if you need another one webview
  4. Activate some other program and press ALT+P to add its window to overlays
  5. Start game
  6. Press ALT+S to show overlays, ALT+H to hide

Advanced

Hotkeys:

  • ALT+Q Quit app

  • ALT+S Show overlays

  • ALT+H Hide overlays

  • ALT+W add Webview overlay

  • ALT+P add overlay for focused Programm

Configure:

Edit settings.cfg file. It text line by line config.

First count of apps overlays. Then each line is a pattern that app use to find running app. For example "otepad.exe" will show overlays for "notepad.exe" and even for "Rotepad.exe" if you have that running.

Then go line with count of webview overlays. Each webview have 5 lines: x, y, widht, height, url
